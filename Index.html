Login

Step 1: Structure of the File

The file has 3 parts:

  • HTML → Structure (inputs, button, container)
  • CSS → Style (center layout, colors, shadow)
  • JavaScript → Logic to check login

Step 2: HTML (Login Form)

<input type="text" id="username">
<input type="password" id="password">
<button onclick="login()">Login</button>

- id="username" & id="password" → JS reads input
- Button → runs login() function
- Message box → shows success/error

Step 3: CSS (Styling)

- Centers the box on screen
- White card with rounded corners & shadow
- Buttons styled with green background & hover effect
- .success = green text, .error = red text

Step 4: JavaScript (Validation)

function login() {
  const user = document.getElementById("username").value;
  const pass = document.getElementById("password").value;
  if(user === "admin" && pass === "12345") {
    msg.textContent = "✅ Login Successful!";
  } else {
    msg.textContent = "❌ Wrong username or password";
  }
}

Checks username & password and shows result message.

Step 5: Security Note

- This is only client-side demo.
- Real websites check login on a server.
- Passwords are stored securely (hashed).
